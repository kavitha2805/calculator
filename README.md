Simple Calculator

A web-based simple calculator built using HTML, CSS, and JavaScript that performs basic arithmetic operations and demonstrates event handling and logical computation in JavaScript.
 Features

Perform basic arithmetic operations (+, −, ×, ÷)

Responsive and clean user interface

Interactive button-based input

Real-time calculation on button click

Clear (C) functionality to reset input

Technologies Used

HTML – Structure and layout of the calculator

CSS – Styling using CSS Grid / Flexbox

JavaScript – Event handling and calculation logic using eval()
